🎨 UX Demonstration Features

Visual Hierarchy

  • Mode Switcher: 1.5px border, shadow, 600 font-weight
  • Theme Toggle: 1px border, 500 font-weight (secondary)
  • Clear Logo → Mode → Theme progression
  • Hover states enhance the hierarchy

Interaction States

  • Hover: Scale 1.02 with background change
  • Active: Primary background, no scale (prevents double-scaling)
  • Focus: Ring outline for keyboard navigation
  • Smooth 200ms transitions throughout

Responsive Behavior

  • Desktop: Full icon + text labels
  • Tablet: Compact text, maintained prominence
  • Mobile: 2×2 grid (spiritual alignment with homepage)
  • Touch-friendly 44px+ targets

Accessibility Features

  • ARIA role="radio" for mode selection
  • Clear focus indicators
  • Keyboard navigation support
  • Reduced motion respect
